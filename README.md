使用了如上的python脚本自动生成 mdbook 所需的 SUMMARY.md 常用的就如下语句：       

> python mdbook-auto-summary.py {absolute path} -o      

其中 -o 是overwrite

方便复制
> python mdbook-auto-summary.py ~/Desktop/cs-notes/src -o
